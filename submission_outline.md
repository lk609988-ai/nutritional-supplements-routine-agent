# Agent Expo 제출서 초안

## 1. Agent 명칭
영양제 루틴 점검 AI

## 2. 제작 목적
여러 종류의 영양제를 복용하는 사용자는 동일 성분의 중복, 적절한 복용 시간,
음식과의 관계, 처방약과의 주의사항을 제품별로 직접 확인해야 한다.
본 Agent는 사용자가 보유한 영양제 제품의 라벨 정보를 한 번에 입력하면
제품 전체를 통합 분석하여 실제로 따를 수 있는 하루 복용 루틴을 제공하기 위해 제작했다.

## 3. 핵심 사용자 문제
- 영양제가 많아 아침·점심·저녁 중 언제 먹을지 알기 어려움
- 멀티비타민과 단일 성분 제품 사이의 중복을 확인하기 어려움
- 새로운 제품을 추가해도 기존 제품과 함께 먹어도 되는지 판단하기 어려움
- 복용 중인 약이나 질환이 있을 때 어떤 항목을 전문가에게 확인해야 하는지 모름

## 4. 주요 기능

### 전체 제품 점검
제품명, 1회 섭취량, 성분과 함량, 라벨 권장량, 현재 복용 시간을 입력받아
제품별로 유지 가능, 시간 조정, 중복 확인, 전문가 확인 필요 여부를 구분한다.

### 중복 성분 분석
여러 제품에 포함된 동일 성분을 묶고 제품별 함량과 계산 가능한 하루 합계를 제시한다.
단위나 개별 함량이 불명확하면 임의로 계산하지 않고 확인 필요로 표시한다.

### 하루 복용 시간표
사용자의 식사, 운동, 카페인 시간을 반영하여 기상 직후, 아침, 점심,
저녁, 취침 전으로 제품을 배치한다. 함께 복용할 수 있는 제품과 간격이 필요한
제품을 별도로 구분한다.

### 제품 추가 비교
사용자가 새로운 제품을 추가하면 기존 제품 목록과 비교하여 중복 성분,
복용 시간과 주의사항이 어떻게 달라지는지 설명한다.

### 안전 중심 응답
라벨 권장량을 넘는 용량, 치료 목적 고용량, 처방약 변경을 제안하지 않는다.
임신·수유, 주요 질환, 수술 예정, 중요한 처방약 복용 시에는 의료진 또는
약사 확인을 우선한다.

## 5. Agent 작동 과정
1. 사용자 건강·생활 정보 수집
2. 보유 영양제 전체 목록 입력
3. 누락된 핵심 정보 최대 3개 질문
4. 제품별 분석
5. 중복 성분과 하루 합계 계산
6. 개인 일정에 맞는 복용 시간표 생성
7. 위험 신호와 전문가 확인 항목 제시
8. 제품 변경 시 기존 루틴과 비교

## 6. 차별점
일반적인 영양제 검색 서비스는 한 제품의 효능이나 복용법만 설명한다.
본 Agent는 사용자가 실제로 보유한 여러 제품을 하나의 조합으로 보고,
중복 성분과 복용 편의성, 생활 시간, 처방약·질환 관련 주의점을 함께 점검한다.
또한 무조건적인 추천이 아니라 불확실한 항목과 전문가 확인 영역을 명확히 구분한다.

## 7. 사용 예시
사용자가 멀티비타민, 비타민 D, 오메가-3, 마그네슘 제품의 라벨 정보를 입력하면
Agent는 비타민 D 등 중복 가능 성분을 확인하고 식사와 함께 먹을 제품,
취침 전 배치할 제품, 약과 간격 확인이 필요한 제품을 구분해 하루 계획으로 제시한다.

## 8. 기대 효과
- 복잡한 복용 계획을 한눈에 확인
- 동일 성분의 불필요한 중복 발견
- 생활 패턴에 맞는 지속 가능한 복용 습관 형성
- 의료진 또는 약사에게 확인할 질문을 사전에 정리

## 9. 한계
본 Agent는 의료 진단이나 처방을 제공하지 않는다.
제품 라벨이 부정확하거나 처방약·질환 정보가 누락되면 완전한 안전성 판단이 어렵다.
고위험 상황에서는 실제 제품 라벨을 지참하고 전문가와 상담해야 한다.
